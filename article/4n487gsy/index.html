一:基本定义

约 531 字大约 2 分钟

2025-05-25

仿射集

最简单的一个凸集, 若一个集合是仿射集, ∀x,y∈C,∀λ∈R,λx+(1−λ)y∈C\forall x,y \in C, \forall \lambda \in \mathbb{R}, \lambda x + (1-\lambda)y \in C∀x,y∈C,∀λ∈R,λx+(1−λ)y∈C 也就是任意两点的直线都在这个集合内.

一个直线就是一个仿射集, 一个二维空间也是一个仿射集, 但是一个圆不是仿射集.

对于取 k 个点, 设 x1,x2,…,xkx_1, x_2, \ldots, x_kx1​,x2​,…,xk​ 是 k 个点, λ1,λ2,…,λk\lambda_1, \lambda_2, \ldots, \lambda_kλ1​,λ2​,…,λk​ 是 k 个实数, 如果满足 ∑i=1kλi=1\sum_{i=1}^k \lambda_i = 1∑i=1k​λi​=1 且 ∀i,λi∈R\forall i, \lambda_i \in \mathbb{R}∀i,λi​∈R, 那么仿射组合∑i=1kλixi\sum_{i=1}^k \lambda_i x_i∑i=1k​λi​xi​ 也在这个集合内.

仿射包

仿射包是一个集合的所有仿射组合的集合, 也就是所有满足 ∑i=1kλi=1\sum_{i=1}^k \lambda_i = 1∑i=1k​λi​=1 且 ∀i,λi∈R\forall i, \lambda_i \in \mathbb{R}∀i,λi​∈R 的点的集合. 其实就是一个集合最小的仿射集, 包含了所有的仿射组合.

凸集 convex set

当线段在集合内时, 这个集合就是凸集, 也就是 ∀x,y∈C,∀λ∈[0,1],λx+(1−λ)y∈C\forall x,y \in C, \forall \lambda \in [0,1], \lambda x + (1-\lambda)y \in C∀x,y∈C,∀λ∈[0,1],λx+(1−λ)y∈C. 仿射集是凸集的一个特例, 但是并不是所有的凸集都是仿射集.

凸组合/ 凸包

凸组合是指满足 ∑i=1kλi=1\sum_{i=1}^k \lambda_i = 1∑i=1k​λi​=1 且0≤λi≤1∑i=1kλi=10 \leq \lambda_i \leq 1 \sum_{i=1}^k \lambda_i = 10≤λi​≤1∑i=1k​λi​=1 的点的集合, 也就是所有非负实数的线性组合. 凸包就是一个集合最小的凸集, 包含了所有的凸组合.

锥

∀xinC,λ≥0\forall x in C, \lambda \geq 0∀xinC,λ≥0, 有 λx∈C\lambda x \in Cλx∈C. 过原点, 然后几条射线过原点, 形成一个锥.

凸锥 convex cone

凸锥是一个锥的特例, 也就是满足 ∀x,y∈C,∀λ≥0,μ≥0\forall x,y \in C, \forall \lambda \geq 0, \mu \geq 0∀x,y∈C,∀λ≥0,μ≥0, 有 λx+μy∈C\lambda x + \mu y \in Cλx+μy∈C 的锥.

凸锥组合/ 凸锥包

凸锥包就是包含一个集合最小的凸锥

几种组合的不同

仿射组合要求是 ∑αi=1\sum \alpha_i = 1∑αi​=1, 且 αi\alpha_iαi​ 可以是负数

凸组合要求是 ∑αi=1\sum \alpha_i = 1∑αi​=1, 且 αi\alpha_iαi​ 是非负数

凸锥组合要求是 αi\alpha_iαi​ 是非负数, 不要求和为1
